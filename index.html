I was bored, so I built a functioning brain in Minecraft

This is a fully functioning image recognition neural network modeled in Minecraft. It's based on how our own brains recognize images. It will "read" what's on the input (retina) and make a guess as to which digit is written. For the most part, it will work with just about any numerical digit that you will visually recognize.

Reverse engineered from this source code and originaly written for an episode of YouTube Premium's MindField.

Training Mr. Squiggles to take over the world.

Here we are training 4 copies of the same AI bot named Mr. Squiggles. Mr. Squiggles doesn't know anything and isn't programmed to do anything. The only difference between each screen is the environment and the goal on which each bot's "fitness" rating is scored:

  1. Top-Left: Distance run to the right (running)
  2. Top-Right: Vertical distance clearned (Jumping)
  3. Bottom-Left: Obstactle avoidance
  4. Bottom-Right: Climbing stairs

In each iteration, 25 Mr. Squiggles bots are generated at the same time (the best one is highlighted). The bots that perform their tasks the best (ie, the "fittest") get to breed. Each new generation, usually, gets a little bit more evolved at soliving their task. This is meant to help visualize what's going on with more robust AI neural networks.

Feel free to skip ahead to see how they all work out. Or, go ahead and make your own bots and see what they can do for yourself!
